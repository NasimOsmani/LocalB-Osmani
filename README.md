LocalBite is an ongoing group project for our **CSE470: Software Engineering** course.
It is a community-focused web application that helps students and local residents share homemade meals, reduce food waste, and connect with nearby home cooks.

This project is being developed using the MERN stack and follows the MVC architecture.
We are currently working on the core features, UI design, and backend structure.

## Project Summary

LocalBite aims to create a friendly student-community platform where anyone can share extra portions of home-cooked food or discover affordable meals nearby.
The platform focuses on:

- Encouraging community bonding
- Reducing food waste
- Offering budget-friendly food options for students
- Supporting local home-based cooks
